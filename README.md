# B2B Customer CRM MVP

Dette er et MVP-prosjekt for en enkel CRM-løsning for B2B-kunder.  
Prosjektet viser ferdigheter i React, CSS og enkel interaktivitet, perfekt for porteføljevisning.

## Teknologier brukt
- React
- CSS
- JavaScript

## Funksjoner
- Enkel login med hardkodet brukernavn og passord
- Dashboard som viser en liste over kunder
- Søkefunksjon for kunder
- Kundekort med statusindikator og hover-effekt
- Responsivt design og enkel UI
